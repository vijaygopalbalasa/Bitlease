Step-by-Step Process

Step 1: Bitcoin Staking (Non-custodial)

Alice visits stake.coredao.org and locks 0.1 BTC via CLTV timelock on Bitcoin mainnet. She retains private keys while Bitcoin is timelocked for 6 months.

# Bitcoin CLTV Script OP_IF <Alice_pub_key> OP_CHECKSIG OP_ELSE <lock_time> OP_CHECKLOCKTIMEVERIFY OP_DROP <CoreDAO_federation_pub_key> OP_CHECKSIG OP_ENDIF

Step 2: bBTC Minting

CoreDAO protocol detects the Bitcoin timelock transaction and mints 0.1 bBTC to Alice's CoreDAO address. This is an ERC-20 receipt token representing her staked Bitcoin plus accrued CORE rewards.

// bBTC.sol (ERC-4626 vault) function mint(bytes32 btcTxHash, uint256 amount) external { require(verifyBitcoinTx(btcTxHash, amount), "Invalid BTC tx"); _mint(msg.sender, amount); }

Step 3: Asset Leasing

Alice visits bitlease.xyz and selects "24h A100 GPU - $60". The platform locks her 0.1 bBTC as collateral and provides $60 USDC at 50% LTV ratio.

// LendingPool.sol function borrow(uint256 usdcAmount, address asset) external { uint256 collateralNeeded = (usdcAmount * 1e18) / (btcPrice * LTV); IERC20(bBTC).transferFrom(msg.sender, address(this), collateralNeeded); IERC20(USDC).transfer(msg.sender, usdcAmount); }

Step 4: Provider Payment

BitLease backend converts USDC to USD and pays the GPU provider (Render Network) via traditional payment rails. Alice receives SSH access credentials and begins using the GPU.

Step 5: Loan Repayment

After using the GPU, Alice repays $60.60 USDC (including 1% origination fee) to unlock her bBTC collateral. Her bBTC has grown to 0.10042 due to CORE staking rewards earned during the lease period.

⚙️ Bitcoin Staking Mechanics

Core Reward Distribution

CORE token emission: CoreDAO follows an 81-year emission schedule with 2.1B total CORE tokens. Daily emission is approximately 28,800 CORE tokens, decaying 3.6% annually.

Reward Calculation Formula

User BTC Weight = BTC_locked / total_BTC_locked User Reward/Day = daily_emission × user_BTC_weight APR = (daily_CORE × 365 × CORE_price) / (BTC_locked × BTC_price) Example (current numbers): - Total BTC locked: 5,250 BTC - Alice locks: 0.1 BTC - Alice share: 0.1 / 5,250 = 0.00001905 - Daily CORE: 28,800 × 0.00001905 = 0.548 CORE/day - USD value: 0.548 × $2.05 ≈ $1.12/day → 5.5% APR

Claiming vs Non-Claiming

Default behavior: CORE rewards automatically compound into bBTC balance every block. Users don't need to claim manually.

Manual claiming: Users can visit stake.coredao.org to claim rewards separately, but this doesn't affect bBTC balance calculation.
